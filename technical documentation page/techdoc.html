Introduction

Python is an interpreted high-level general-purpose programming language. It was created by Guido van Rossum, and released in 1991. It is used for server-side web development, software development, mathematics, data science, and scripting.

Python was designed for readability, and has some similarities to the English language with influence from mathematics. relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

Variables

Variables are names given to certain locations in memory i.e. storing data/variables using names or labels. Unlike other languages like Javascript, Python has no command for declaring varibales.

>>> greeting = "Hello World!" >>> print(greeting) Hello World!

There are rules to follow when creating variables in Python.

  • A variable name can't start with a number.
  • A variable name can only contain letters, numbers and underscores e.g. >>> greet = "Hello" Hello >>> greet2 = "Hi" Hello >>> _greet = "¡Hola!" ¡Hola!
  • A variable name can't be a keyword.
  • A variable name can't contain spaces e.g. >>> greet 2 = "Hi" File "", line 1 greet 2 = "Hi" ^ SyntaxError: invalid syntax This variable name will result in an error.
  • A variable name can not start with a number.

Python also doesn't require declaring the data type of the value.

a = "Hi!" # a is a string. b = 43 # b is an integer. c = 43.0 # c is a float

Variable Scope

Scope is the part of the program where a variable can be accessed. Not all variables created can be accessed from anywhere in a program. There are four major types of scope:

  • Local scope: When a variable is declared, it is accessible from then until the end of the program. It is available for as long as the program is running. This is true especially when it is created in a function. It will be accessible only within the function.
  • Enclosing scope: This scope applies to nested functions. When a variable is declared in the first(outer) function, it can be accessed by the second(inner) function but a variable declared in the second function can't be accessed by the first function.
  • Global scope: When a function is declared outside a function, it can be accessed by any function for instance if a variable is declared before the first nested function, it can be accessed by both the first and second function.
  • Built-in scope: This applies to reserved keywords. Keywords are words reserved by Python for specific purposes and can't be used for anything else.

Data Types

Every value in Python has a data type. The data type tells the interpretor how that data is to be used. Python has various data types and variables can store different data types.

int
Integers can be any length as long as the memory allows it. They are designated to the 'int' type.

>>> print(806) 806 >>> print(75645238469308470572092734091034230472304027304) 75645238469308470572092734091034230472304027304

float
Floating-point numbers are values specified by a decimal point. They are designated to the 'float' type. In Python, floats can be accurate by upto 15 decimal places.

>>> print(8.06) 8.06 >>> print(5423764971832481034.6834572903126341) 5.423764971832481e+18

complex
Complex numbers are numbers in the form 'a+bj', where a is the real part, b is the imaginary part and j is the imaginery unit. These numbers are especually useful in electronics, optics and quantum theory.

>>> print(23+56j) (23+56j)

str
Strings are a sequences of character data or Unicode characters. Strings are denoted with '' or "" and are designated to the 'str' type.

>>> print("Strings look like this.") Strings look like this.

Like integers, strings can be as long as you want, only constrained by memory.

Setting data types
Data types can be set when declaring variables as seen below:

>>> age = int(23) >>> w = str("The wrong way.")

To find out the data type of a value, 'type()' is used.

>>> w = str("The wrong way.") >>> type(w)
